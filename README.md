# FatalForce
📌 Project Overview

This project explores a highly sensitive and complex issue: fatal police shootings in the United States. Using data collected since January 1, 2015, we analyze patterns and potential relationships between police shootings and socioeconomic factors such as:

Poverty Rate

High School Graduation Rate

Median Household Income

Racial Demographics

The goal is to better understand underlying social trends and contribute to data-driven discussions.

📂 Dataset Sources

Police Shooting Data: Washington Post Fatal Force Dataset

Socioeconomic Data: US Census Data

Files Included:

fatal_police_shootings.csv

poverty_rate.csv

median_income.csv

high_school_graduation.csv

racial_demographics.csv

analysis.ipynb (Jupyter Notebook)

Python 🐍

Pandas (Data Cleaning & Manipulation)

🔍 Key Steps in Analysis

Data Cleaning

Handling missing values (-, NaN)

Converting data types (e.g., string → numeric)

Standardizing state names and codes

Data Merging

Combined datasets using state-level keys

Ensured consistency across different data sources

Exploratory Data Analysis (EDA)

Distribution of shootings by race, age, gender

Year-wise trends

State-wise comparison

Visualization

Bar charts, box plots, and choropleth maps

Correlation plots between socioeconomic variables and shootings

📈 Key Insights

Certain states show higher frequencies of fatal police shootings.

Socioeconomic indicators like poverty and income show some correlation trends, but not causation.

Racial distribution highlights disparities that require deeper contextual understanding.
